## Getting started

Clone the repo

```
git clone git@github.com:devsoc-unsw/trainee-hades-26t1.git
```
or
```
git clone https://github.com/devsoc-unsw/trainee-hades-26t1.git
```

## Frontend

1. Setting up frontend

```
cd frontend
pnpm install
```
2. Create `.env` file. Paste the content in `docs-and-resources` from our discord server.
3. Running frontend

```
pnpm dev
```

## Backend

1. Setting up backend

```
cd backend
pnpm install
```
2. Create `.env` file. Paste the content in `docs-and-resources` from our discord server.
3. Running backend

```
pnpm dev
```
